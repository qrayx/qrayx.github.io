qrayx's ChemSpell Checker

This tool is used to figure out which words can be spelled with elemental symbols (e.g. "H" for "Hydrogen", or "Fe" for "Iron"). Type in any list of words into the input and click the Check button. Try it with the examples given.

If a word has multiple ways of being spelled with symbols, each possibility will be printed.

You can toggle the output to print the elemental symbols (e.g. "W") or the full names of elements (e.g. "Tungsten").

Only letters will be considered. There is no case sensitivity. Non-letters (numbers, symbols, spaces) will be ignored.
